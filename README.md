# Employee Management System (HRdocs)

## First Release

This is the first official release of the Employee Management System.
It is a desktop application designed to manage employees, departments,
absences, and certificates across multiple cities.

The application is distributed as a Windows installer (.exe).

---

## Features

### Employee Management
- Add and delete employees
- Assign employees to departments and cities
- Manage employees per department and per city

### Absence Management
- Manage employee absences for each year
- Track yearly absence records per employee

### Certificate Generation
- Generate ready-to-use certificates for employees
- One-click certificate creation with employee information

### Search and Filtering
- Search across all employees
- Filter employees by department, city, and year

### Employee History
- Track all generated certificates

### File Management
- Automatically save generated files and certificates
- Centralized document storage

---

## Download and Installation (Windows)

### Download
1. Go to the Releases page of this repository.
2. Download the latest .exe file from the Assets section.

### Install
1. Double-click the downloaded .exe file.
2. Start the application from the Desktop shortcut or Start Menu.

No additional configuration is required.
